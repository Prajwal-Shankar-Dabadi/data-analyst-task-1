# Data Analyst Internship – Task 1  
## Excel Data Cleaning & Formatting

### Objective
Clean and standardize a real-world dataset using Excel by handling missing values, duplicates, and inconsistent formatting.

### Dataset
**Netflix Movies and TV Shows Dataset**

### Tools Used
- Microsoft Excel

### Key Cleaning Steps
- Preserved original data as `Raw_Data.xlsx`
- Removed duplicate records using key identifiers
- Handled missing values in important columns (director, country)
- Standardized text fields and formats
- Converted date columns to proper date format
- Added `Data_Quality_Notes` to document issues

### Files Included
- `Raw_Data.xlsx`
- `Cleaned_dataset.xlsx`
- `cleaned_dataset.csv`

### Outcome
A clean, structured, and analysis-ready dataset following professional data cleaning practices.
